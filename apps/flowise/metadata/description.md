# Flowise

Open-source low-code builder for **LLM apps and AI agents**. Visually chain LLMs, tools, memory, and vector stores, then serve them as APIs.

- Image: `flowiseai/flowise`
- Internal port: `3000`
- Admin username/password are set at install time; data persists under the app data directory.

> China/starway tip: if `docker.io` pull is slow/blocked, mirror `flowiseai/flowise:2.2.2` to your Aliyun ACR and update the `image:` line.
